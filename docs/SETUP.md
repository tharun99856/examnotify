## API Endpoints

| Method | Path         | Description           |
|--------|--------------|-----------------------|
| GET    | /api/v1/data | Retrieve records      |
| POST   | /api/v1/data | Create a new record   |
| PATCH  | /api/v1/data/:id | Update a record   |
| DELETE | /api/v1/data/:id | Remove a record   |
